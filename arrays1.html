Spread operator:- ---------------------- Spread operator allows an iterable to expand in palces where 0+ arguments are expected. It is mostly used in the variable array where there is more than 1 values are expected. It allows us the privilege to obtain a list of parameters from an array. syn: var variablename1=[...value]; ... -> spread operator Rest parameter:- -------------------- Rest parameter is an improved way to handle function parameter, allowing us to more easily handle various input as parameters in a function. syn: function functionname[...parameters] { statement; }
